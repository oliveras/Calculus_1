Trig Identities

\(\displaystyle\cot(\theta) = \frac{\cos(\theta)}{\sin(\theta)}\) \(\displaystyle \csc(\theta) = \frac{1}{\sin(\theta)}\)
\(\displaystyle \sec(\theta) = \frac{1}{\cos(\theta)}\) \(\displaystyle \sin(-\theta) = -\sin(\theta)\), \(\displaystyle \cos(-\theta) = \cos(\theta)\)
\(\displaystyle \sin^2(\theta) + \cos^2(\theta) = 1\) \(\displaystyle \tan^2(\theta) + 1 = \sec^2(\theta)\)
\(\displaystyle 1+\cot^2(\theta) = \csc^2(\theta)\) \(\displaystyle \tan(\theta) = \frac{\sin(\theta)}{\cos(\theta)}\)
\(\displaystyle\cot(\theta) = \frac{\cos(\theta)}{\sin(\theta)}\) \(\displaystyle \csc(\theta) = \frac{1}{\sin(\theta)}\)
\(\displaystyle \sec(\theta) = \frac{1}{\cos(\theta)}\) \(\displaystyle \sin(-\theta) = -\sin(\theta)\)
\(\displaystyle \cos(-\theta) = \cos(\theta)\) \(\displaystyle \sin(2\theta) = 2\sin(\theta) \cos(\theta)\)
\(\displaystyle \cos(2\theta) = \cos^2(\theta) - \sin^2(\theta)\) \(\displaystyle \cos(2\theta) = 2\cos^2(\theta) - 1 \)
\(\displaystyle \cos(2\theta)= 1 - 2\sin^2(\theta)\) \(\displaystyle \sin^2(\theta) = \frac{1-\cos(2\theta)}{2}\)
\(\displaystyle \cos^2(\theta) = \frac{1+\cos(2\theta)}{2}\) \(\displaystyle e^{\pm i \theta} = \cos\theta \pm i \sin\theta, \quad i = \sqrt{-1}\)
\(\displaystyle \cos(\theta) = \frac{1}{2}\left(e^{i\theta} + e^{-i\theta}\right), \quad i = \sqrt{-1}\) \(\displaystyle \sin(\theta) = \frac{1}{2i}\left(e^{i\theta} + e^{-i\theta}\right), \quad i = \sqrt{-1}\)
\(\displaystyle \sin(a)\cos(b) = \frac{1}{2}\left(\sin(a-b) + \sin(a + b)\right)\) \(\displaystyle \sin(a)\sin(b) = \frac{1}{2}\left(\cos(a-b) - \cos(a + b)\right)\)
\(\displaystyle \cos(a)\cos(b) = \frac{1}{2}\left(\cos(a-b) + \sin(a + b)\right)\) \(\displaystyle \sin(a + b) = \sin(a)\cos(b) + \cos(a)\sin(b)\)
\(\displaystyle \cos(a + b) = \cos(a)\cos(b)-\sin(a)\sin(b)\)

 

Hyperbolic Trig Identities

\(\displaystyle \cosh^2(\theta) - \sinh^2(\theta) = 1\) \(\displaystyle 1 -\tanh^2(\theta) = \sech^2(\theta)\)
\(\displaystyle \coth^2(\theta)-1 = \csch^2(\theta)\) \(\displaystyle \tanh(\theta) = \frac{\sinh(\theta)}{\cosh(\theta)}\)
\(\displaystyle\coth(\theta) = \frac{\cosh(\theta)}{\sinh(\theta)}\) \(\displaystyle \csch(\theta) = \frac{1}{\sinh(\theta)}\)
\(\displaystyle \sech(\theta) = \frac{1}{\cosh(\theta)}\) \(\displaystyle \sinh(-\theta) = -\sinh(\theta)\)
\(\displaystyle \cosh(-\theta) = \cosh(\theta)\) \(\displaystyle \sinh(2\theta) = 2\sinh(\theta) \cosh(\theta)\)
\(\displaystyle \cosh(2\theta) = \cosh^2(\theta) + \sinh^2(\theta)\) \(\displaystyle \cosh(2\theta) = 2\cosh^2(\theta) - 1 \)
\(\displaystyle \cosh(2\theta)= 2\sinh^2(\theta) + 1\) \(\displaystyle \sinh^2(\theta) = \frac{\cosh(2\theta)-1}{2}\)
\(\displaystyle \cosh^2(\theta) = \frac{1+\cosh(2\theta)}{2}\) \(\displaystyle e^{\pm\theta} = \cosh(\theta) \pm \sinh\theta\)
\(\displaystyle \cosh(\theta) = \frac{1}{2}\left(e^\theta + e^{-\theta}\right)\) \(\displaystyle \sinh(\theta) = \frac{1}{2}\left(e^\theta - e^{-\theta}\right)\)

 

Derivatives of Trig Functions and Hyperbolic Trig Functions

\(\displaystyle \frac{d}{dx}\left[\sin(x)\right] = \cos(x)\) \(\displaystyle \frac{d}{dx}\left[\sin^{-1}(x)\right] = \frac{1}{\sqrt{1- x^2}},\qquad \vert x \vert <1\)
\(\displaystyle \frac{d}{dx}\left[\cos(x)\right] = -\sin(x)\) \(\displaystyle \frac{d}{dx}\left[\cos^{-1}(x)\right] = \frac{-1}{\sqrt{1 - x^2}},\qquad \vert x \vert < 1\)
\(\displaystyle \frac{d}{dx}\left[\tan(x)\right] = \sec^2(x)\) \(\displaystyle \frac{d}{dx}\left[\tan^{-1}(x)\right] = \frac{1}{1+x^2}, \qquad x \in\mathbb{R}\)
\(\displaystyle \frac{d}{dx}\left[\sinh(x)\right] = \cosh(x)\) \(\displaystyle \frac{d}{dx}\left[\sinh^{-1}(x)\right] = \frac{1}{\sqrt{1 + x^2}},\qquad x\in\mathbb{R}\)
\(\displaystyle \frac{d}{dx}\left[\cosh(x)\right] = \sinh(x)\) \(\displaystyle \frac{d}{dx}\left[\cosh^{-1}(x)\right] = \frac{1}{\sqrt{x^2-1}}, \qquad \vert x \vert > 1\)
\(\displaystyle \frac{d}{dx}\left[\tanh(x)\right] = \sech^2(x)\) \(\displaystyle \frac{d}{dx}\left[\tanh^{-1}(x)\right] = \frac{1}{1 - x^2}, \qquad \vert x\vert < 1\)
