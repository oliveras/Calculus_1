\(\newcommand{\derRule}[2]{\displaystyle  \frac{d}{dx}\left[#1\right] = #2}\)

This page will contain information about the derivative, and properties of the derivative.  You can also find complete list in the textbook's Table of Derivatives

Properties of the Derivative

Derivative - Limit
Definition (3.2)
\(\derRule{f(x)}{\lim_{h\to 0}\frac{f(x + h)-f(x)}{h}}\)
Sum/Difference
Rule (3.3)
\(\derRule{f(x)\pm g(x)}{f'(x)\pm g'(x)}\).
Constant
Multiple Rule (3.3)
\(\derRule{k\cdot g(x)}{k \cdot g'(x)}\) where \(k\) is a constant 
Implicit Diff.
(3.8)
Example: \(\displaystyle \frac{d}{dx}\left[x^2 + y^2\right] = \frac{d}{dx}\left[x^2 + \left(y(x)\right)^2\right] = 2 x + 2 y(x)\cdot y'(x)\)

 

Rules for Differentiation 

Constant Function
(3.3)
\(\derRule{c}{0}\) where \(c\) is a constant.
Power Rule
(3.3)
\(\derRule{x^n}{nx^{n-1}}\).
Product
Rule (3.3)
\(\derRule{f(x)\cdot g(x)}{f'(x)\cdot g(x) + f(x)\cdot g'(x)}\)
Quotient
Rule (3.3)
\(\derRule{\frac{f(x)}{g(x)}}{\frac{f'(x)\cdot g(x) - f(x)\cdot g'(x)}{\left(g(x)\right)^2}}\)
Chain
Rule (3.6)
\(\derRule{f(g(x))}{f'(g(x))\cdot g'(x)}\)
Extended Power
Rule (3.6)
\(\derRule{\left(g(x)\right)^n}{n\left(g(x)\right)^{n-1}\cdot g'(x)}\)
Exponential
Rules (3.9)
\(\derRule{e^x}{e^x}\)

\(\derRule{a^x}{\ln(a)a^x}\)
Extended Exp
Rules (3.9)
\(\derRule{e^{g(x)}}{e^{g(x}g'(x)}\)

\(\derRule{a^{g(x)}}{\ln(a)a^{g(x)}g'(x)}\)
Logarathim
Rules (3.9)
\(\derRule{\ln(x)}{\frac{1}{x}}\)

\(\derRule{\log_a(x)}{\frac{1}{x\ln(a)}}\) 
Extended Log
Rules (3.9)
\(\derRule{\ln(g(x))}{\frac{g'(x)}{g(x)}}\)

\(\derRule{\log_a(g(x))}{\frac{g'(x)}{g(x)\ln(a)}}\) 

Derivatives of Trig Functions

Sine
(3.5)
\(\derRule{\sin(x)}{\cos(x)}\).
Cosine
(3.5)
\(\derRule{\cos(x)}{-\sin(x)}\).
Tangent
(3.5)
\(\derRule{\tan(x)}{\sec^2(x)}\).
Secant
(3.5)
\(\derRule{\sec(x)}{\sec(x)\tan(x)}\).
