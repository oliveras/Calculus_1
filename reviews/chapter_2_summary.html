A FLOW CHART FOR LIMITS

Assume that \(f(x)\) is a function of the form considered in class (polynomial, rational, continuous, etc).  In order to determine \[\displaystyle \lim_{x\to a} f(x),\]
  • If \(a\) is finite, try plugging in the value \(x = a\) into the function \(f(x)\).  If
    • CASE 1: \(f(a) = L\) where \(L\) is a finite number, and \(f(x)\) is continuous at \(x = a\), then \[\displaystyle \lim_{x\to a} f(x) = f(a),\]
    • CASE 2: \(f(a)  = \frac{L}{0}\) where \(L\) is a nonzero, finite number, then \(\displaystyle \lim_{x\to a} f(x)\) either does not exist, or is infinite.    You should always verify an answer of this form by examining a graph of the function near \(x = a\) or by creating a table of values for various \(x\) values near \(x = a\).
    • CASE 3: \(f(a) = \frac{0}{0}\), then you have an indeterminant form ðŸš©.  In order to determine the limit, you should try to factor and simplify to determine the limit by either (1) factoring and canceling, (2) multiplying by the conjugate, or (3) simplifying any complex fractions.  If you get stuck at this point, you can always examine the graph or create a table of values.
  • If \(a\) is infinite, for now, use the graph or create a table of values to determine the limit. We will learn more advanced techniques later in Section 4.8.

 

Topics Covered

The exam will cover sections 1.1, 2.2, 2.3, 2.4, and 2.5 from the book.  The follow is a list of topics that may be help to review.

Section 1.1: Functions

At the end of this section, you should be able to 

  • Know how to represent functions three different ways: 
    • as a graph
    • as a table of values
    • as a formula
  • Simplify expressions such as \(f(x + 2)\) given a definition for \(f(x)\). 
  • Compute the composition of two functions \(f(x)\) and \(g(x)\) as \(f\circ g = f(g(x))\)
  • Suggested Problems: Section 1.1: 7-17, 36, 37, 45, 48, 50, 51

Section 2.2: The Limit of a Function

At the end of this section, you should be able to 

  • Understand what the limit of a function represents and that limits describe the behavior of a function near a value and not at the value.
  • Know how to use a graph and a table of values to estimate a limit. 
  • Understand the difference between the following notations: 
    • \(\displaystyle \lim_{x\to a}f(x)\)
    • \(\displaystyle \lim_{x\to a^+}f(x)\)
    • \(\displaystyle \lim_{x\to a^-}f(x)\)
  • Understand that if the limit \(\displaystyle \lim_{x\to a}f(x)\) does not exist, that \(\displaystyle \lim_{x\to a^+}f(x)\) and \(\displaystyle \lim_{x\to a^-}f(x)\) may exist.
  • How to use limits to describe infinite behavior of a function as \(x\) approaches a certain value.  
  • Suggested Problems: Section 2.2: 30 - 36, 42, 43, 50-54, 79-82

Section 2.3: The Limit Laws

At the end of this section, you should be able to 

  • know how to use the limit laws to evaluate the limit of a function.
  • know that if \(f(x)\) is a polynomial, then \(\displaystyle \lim_{x\to a}f(x) = f(a)\). 
  • know what the indeterminant form \(\frac{0}{0}\) is (and isn't) telling you
  • evaluate the limit by using the following tricks: 
    • factoring and canceling
    • multiplying by the conjugate
    • simplifying a complex fraction
  • state the squeeze theorem in your own words
  • use the squeeze theorem to find the limit of a function if the function is always greater than one function and less than another function with limits that are known and equal.  
  • Suggested Problems: Section 2.3: 83-84, 89-92, 96-99, 103-106, 118-127

Section 2.4: Continuity

At the end of this section, you should be able to 

  • know how to determine if a function \(f(x)\) is continuous at a point \(x = a\) by checking if all three of the following conditions are satisfied:
    • \(f(a)\) exists and is finite
    • \(\displaystyle \lim_{x \to a}f(x)\) exists and is finite
    • \(\displaystyle \lim_{x \to a}f(x) = f(a)\)
  • state the intermediate value theorem in your own words
  • use the intermediate value theorem to show the existence of a solution \(x = c\) to an equation of the form \(f(c) = z\).  
  • Suggested Problems: Section 2.4: 131-135, 139-142, 151, 152, 159-162

Section 2.5: The Precise Definition of the Limit

At the end of this section, you should be able to 

  • understand how the intuitive definition of a limit can be converted into a rigorous mathematical definition known as the \(\epsilon\)-\(\delta\) definition of the limit ("epsilon-delta")
  • Know how to solve problems similar to problem 180 from section 2.5.  
  • Suggested Problems: Section 2.5: 176-181, 188-191
