Section 4.3: Maxima and Minima

At the end of Section 4.3, you should

  • know what an absolute extrema of a function is, as well as the various scenarios that can occur.  See the image below for details.
  • know what a local extrema of a function is.
  • know that if a function has a local extrema, it must occur at a critical number. 
  • know how to compute critical numbers
    We say that \(c\) is a critical number of a function if either \(f'(c) = 0\) or \(f'(c)\) is undefined.  We call the point \((c,f(c))\) a critical point of \(f\).
  • know that a continuous function over a closed, bounded interval has an absolute maximum and an absolute minimum.  The location of an absolute extrema occurs at endpoints and/or at critical points.
  • know how to compute the absolute extrema of a function \(f\) on a closed interval \(\left[a,b\right]\) via the following steps:

    To compute absolute extrema of a function \(f\) on a closed interval \(\left[a, b\right]\), 

    • Step 1: Evaluate \(f\) at the endpoints \(x = a\) and \(x = b\).
    • Step 2: Find all of the critical points of \(f\) that lie in the open interval \((a,b)\) and evaluate \(f\) at those critical values.
    • Step 3: Compare all of the values found in Steps 1 and 2. The largest of the \(f\) values corresponds to absolute maxima, and the smallest of the \(f\) values corresponds to absolute minima.
  • Suggested Problems:

Section 4.4: The Mean Value Theorem

At the end of this Section 4.4, you should 

  • be able to state Rolle's Theorem in your own words and understand the implications.
    Rolle's Theorem: If \(f\) is (1) continuous over \(\left[a,b\right]\), (2) differentiable over \((a,b)\), and (3) \(f(a) = 0 = f(b)\), then there exists a point \(c\in(a,b)\) such that \(f'(c) = 0\).
  • be able to state the Mean Value Theorem in your own words and understant the implications.
    Mean Value Theorem: If \(f\) is continuous over \(\left[a,b\right]\) then there exists a point \(c\in(a,b)\) such that \[f'(c) = \frac{f(b) - f(a)}{b-a}.\]
  • remember how the sign of the derivative \(f'\) is related to the behavior of \(f\).
    • if \(f'(x)>0\) over an interval, then \(f(x)\) is increasing over that interval
    • if \(f'(x)<0\) over an interval, then \(f(x)\) is decreasing over that interval

Section 4.5: Derivatives and the Shape of a Graph

At the end of Section 4.5, you should

  • remember how the sign of the derivative \(f'\) is related to the behavior of \(f\).
    • if \(f'(x)>0\) over an interval, then \(f(x)\) is increasing over that interval
    • if \(f'(x)<0\) over an interval, then \(f(x)\) is decreasing over that interval
  • know how the sign of the second derivative \(f''\) is related to the behavior of \(f\).
    • if \(f''(x)>0\) over an interval, then \(f(x)\) is concave up over than interval.
    • if \(f''(x)<0\) over an interval, then \(f(x)\) is concave down over that inverval.
  • know the difference between concave up and concave down
  • know the first derivative test and the second derivative test and how to use them (example: Video: Section 4.5 - Example: Using the First and Second Derivative Test)

    First Derivative Test: Suppose that \(c\) is a critical number of a continuous function \(f\).

    1. If \(f'(x)\) changes from negative to positive at \(x = c\), then \(f(x)\) has a local minimum at \(x = c\). 
    2. If \(f'(x)\) changes from positive to negative at \(x = c\), then \(f(x)\) has a local maximum at \(x = c\). 
    3. If \(f'(x)\) does not change signs at \(x = c\), then \(f(x)\) does not have a local extrema at \(x = c\). 

    Second Derivative Test: Suppose that \(c\) is a critical number of a continuous function \(f\).

    1. If \(f'(c) = 0\) and \(f''(c)>0\), then \(f(x)\) has a local minimum at \(x = c\). 
    2. If \(f'(c) = 0\) and \(f''(c)<0\), then \(f(x)\) has a local maximum at \(x = c\). 
    3. If \(f'(c) = 0\) and \(f''(c)=0\), then you must fall back to the first-derivative test to draw any conclusions.
  • Suggested Problems:
    • Section 4.5: 194 - 202, 206-208, 216, 228-230, 232-234, 246-250 (see Solution: Section 4.5 Problem 232)

Section 4.6: Limits at Infinity and Asymptotes

At the end of this Section 4.6, you should 

  • know that the limit of \(f(x)\) is \(L\) as \(x \to \infty\) (or as \(x \to -\infty\)) if the values of \(f(x)\) become arbitrarily close to \(L\) as \(x\) becomes sufficiently large.
  • know that the limit of \(f(x)\) is \(\infty\) as \(x \to \infty\) (or as \(x \to -\infty\)) if the values of \(f(x)\) become arbitrarily large as \(x\) becomes sufficiently large. 
  • For a polynomial function \[p(x) = a_nx^n + a_{n-1}x^{n-1} + \ldots+a_1 x + a_0\] where \(a_n\neq 0\), the end behavior is determined by the leading term \(a_nx^n\). If \(n\neq 0\), \(p(x)\) approaches \(\pm \infty\) at each end.
  • know the definition of a rational function and that it can be expressed in the general form  \[f(x) = \frac{p(x)}{q(x)} = \frac{ a_nx^n + a_{n-1}x^{n-1} + \ldots+a_1 x + a_0}{ b _mx^m+ b_{m-1}x^{m-1} + \ldots+b_1 x + b_0}\]
  • know that for a rational function of the form above, the end behavior is determined by the relationship between the degree of \(p\) and the degree of \(q\). 
    • If the degree of \(p\) is less than the degree of \(q\), the line \(y = 0\) is a horizontal asymptote for \(f\). 
    • If the degree of \(p\) is equal to the degree of \(q\), then the line \(y = \frac{a_n}{b_n}\) is a horizontal asymtptoe. 
    • If the degree of \(p\) is greater than the degree of \(q\), then \(f\) approaches either \(\pm\infty\) at each end.

Section 4.8: L'Hopital's Rule

At the end of this Section 4.8, you should 

  • be able to identify various indeterminant forms
  • understand how L’Hôpital’s rule can be used to evaluate the limit of a quotient when the indeterminate form \(\frac{0}{0}\) or \(\frac{\infty}{\infty}\) arises. 
    L’Hôpital’s Rule (0/0 Case)
    Suppose \(f\) and \(g\) are differentiable functions over an open interval containing \(a\) (except possibly at \(x = a\)). If \(\lim_{x\to a} f(x) = 0\) and \(\lim_{x\to a}g(x) = 0\), then \[\lim_{x\to a}\frac{f(x)}{g(x)} = \lim_{x\to a}\frac{f'(x)}{g'(x)},\]assuming the limit on the right exists or is \(\pm \infty\).  This result also holds if we are considering one-sided limits, or if \(a = \pm \infty\). 

    L’Hôpital’s Rule (\(\infty\)/\(\infty\) Case)
    Suppose \(f\) and \(g\) are differentiable functions over an open interval containing \(a\) (except possibly at \(x = a\)). If \(\lim_{x\to a} f(x) = \infty\) (or \(-\infty\)) and \(\lim_{x\to a}g(x) = \infty\) (or \(-\infty\)), then \[\lim_{x\to a}\frac{f(x)}{g(x)} = \lim_{x\to a}\frac{f'(x)}{g'(x)},\]assuming the limit on the right exists or is \(\pm \infty\).  This result also holds if the limit is infinite, we are considering one-sided limits, or if \(a = \pm \infty\). 
  • understand how L’Hôpital’s rule can also be applied to other indeterminate forms if they can be rewritten in terms of a limit involving a quotient that has the indeterminate form \(\frac{0}{0}\) or \(\frac{\infty}{\infty}\).
  • know that the exponential function \(e^x\) grows faster than any power function \(x^p\) where \(p>0\)
  • know that the logarithmic function \(\ln(x)\) grows slower than any power function \(x^p\) where \(p>0\)

Section 4.9: Newton's Method

At the end of Section 4.9, you should

  • know that Newton's method allows you to approximate roots of the equation \(f(x) = 0\) by starting with an initial guess \(x_0\), then uses tangent lines to the graph of \(f(x)\) to create a sequence of approximations \(x_1\), \(x_2\), ...
  • know that Newton’s method is an efficient method for finding a particular root. In certain cases, Newton’s method fails to work because the list of numbers  \(x_0\), \(x_1\), \(x_2\), ...  does not approach a finite value or it approaches a value other than the root sought. 
  • know that Newton’s method is an example of an iterative process, where updated estimates of the root are given by \[x_n = x_{n-1} - \left[\frac{f(x_{n-1})}{f'(x_{n-1})}\right]for a given function \(f(x)\).

Section 4.10: Antiderivatives

At the end of Section 4.10, you should

  • know that finding antiderivatives is "going backwards".  That is, fill in the blank: \[\frac{d}{dx}\big[\underline{~~~~}\big] = 2x.\]In this example, we would fill in the blank with a function such as \(x^2 + c\) where \(c\) is any arbitrary constant.  Thus, \(x^2 + c\) is the antiderivative of \(2x\).
  • know that \(F\) is an antiderivative of a function \(f\) if \(F'(x) = f(x)\).  For example, \(F(x) = \sin(x) + 2x^3\) is the antiderivative of \(f(x) = \cos(x) + 6x^2\) because \[F'(x) = \frac{d}{dx}\left[\sin(x) + 2x^3\right] = \cos(x) + 6x^2 = f(x).\]
  • know that if \(F\) is an antiderivative of \(f\), then every antiderivative of \(f\) is of the form \(F(x) + c\) for some arbitrary constant \(c\). That is, to find the most general antiderivative, you must include an arbitrary constant.
  • know that the integral symbol is used to define an indefinite integral and that the indefinite integral describes the most general antiderivative for a function - that is, \(\int f(x)\,dx = F(x) + c\) where \(F'(x) = f(x)\).  For example, \[\int 2x\,dx = x^2 + c\]We can check the answer by computing \(\frac{d}{dx}\left[x^2 + c\right] = 2x + 0 = f(x)\). 
  • know that our table of derivatives can be inverted to create a table of indefinite integrals.  For example: 
    Power Rule   \(\derRule{x^n}{nx^{n-1}}\) Power Rule \(\intRule{x^n}{\frac{x^{n+1}}{n+1}}\)
    Sine Rule   \(\derRule{\sin(x)}{\cos{x}}\) Sine Rule \(\intRule{\sin(x)}{-\cos(x)}\)
    Cosine Rule   \(\derRule{\cos(x)}{-\sin(x)}\) Cosine Rule \(\intRule{\cos(x)}{\sin(x)}\)
  • know how to solve initial value problems.  That is, given \[\frac{dy}{dx} = f(x), \quad y(x_0) = y_0,\] then \(y(x)\) takes the form \(y(x) = F(x) + c\) where \(F(x)\) is an antiderivative of \(f(x)\) (\(F'(x) = f(x)\).  Using \(y(x_0) = y_0\), you can solve for the arbitrary constant \(c\). 
    • Example: to find the antiderivative of \(\frac{df}{dx} = 2x\) that goes through the point (f(2) = 1\), we know that \(f(x) = x^2 + c\).  Using that \(f(2) = 1\), we have \[f(x) = x^2 + c \quad \to \quad 1 = (2)^2 + c\]which implies that \(c = -3\).  Thus, the solution to the initial value problem is \[f(x) = x^2 - 3.\]  
  • Suggested Problems:
    • Section 4.10: 466-478, 482, 490 - 494, 523 - 524
